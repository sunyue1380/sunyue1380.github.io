介绍

QuickDAO

QuickDAO是一款简单,易用,轻量级的java ORM框架.摒弃MyBatis繁杂的xml文件和mapper文件,吸收hibernate自动建表和外键关联支持,最终精炼成QuickDAO简单易用的API.

特点

  • 强大的外键关联支持,从关联查询到结果返回,轻松应对,从此不用为外键关联查询烦恼
  • 全程Java代码,不涉及任何xml文件
  • 抛弃所有Mapper类和xml配置文件,只需要一个dao对象即可完成数据库所有操作
  • 自动建表,自动新增字段
  • 内置数据库方言支持,不用考虑数据库方言问题
  • 实体注解支持,可直接在实体类字段上定义字段类型,唯一性约束,索引等等,从此不用跟数据库打交道

支持数据库

  • MySQL(5.0以上)
  • SQLite
  • H2
  • Postgre(9.0.0以上)
  • SQL Server(2012版本以上)

快速体验

QuickDAO使用起来非常方便,它到底有多方便呢? talk is cheap,show me the code!

  • 根据id查询

    User user = dao.fetch(User.class,1);

  • 根据name属性查询

    List<User> userList = dao.fetchList(User.class,"name","quickdao");

  • 插入删除实体对象

    //插入
dao.insert(user);
//更新
dao.update(user);
//存在则更新,不存在则插入
dao.save(user);

  • 单表复杂查询

    List<User> userList = dao.query(User.class)
  .addQuery("name","quickdao")
  .addNotNullQuery("password")
  .page(1,10)
  .orderBy("id")
  .execute()
  .getList();

  • 外键关联查询

    List<User> userList = dao.query(User.class)
  joinTable(Address.class,"addressId","id")
  .addQuery("name","BeiJing")
  .done()
  .addQuery("name","quickdao")
  .page(1,10)
  .orderBy("id")
  .compositField()
  .execute()
  .getList();
